# adobe-launch-exporter


Privacy Policy for Adobe Launch Exporter Chrome extension

1. Data Collection: This extension collects Adobe Launch API credentials (Client ID, Secret, Org ID) provided by the user.
2. Data Usage: These credentials are used solely to authenticate requests to the Adobe Experience Platform API.
3. Data Storage: All credentials and retrieved container data are stored locally on the user's device using chrome.storage.local. No data is ever transmitted to the developer or any third-party servers.
4. Third Parties: Data is only shared with Adobe Systems via their official API endpoints to fulfill the extension's purpose.
